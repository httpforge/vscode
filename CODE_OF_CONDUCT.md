# Contributor Covenant Code of Conduct

## Our Pledge

We as members, contributors, and maintainers pledge to make participation in our community a harassment-free experience for everyone.

## Expected Behavior

* Be respectful
* Be welcoming
* Be constructive
* Be collaborative
* Accept feedback gracefully
* Focus on what is best for the community

## Unacceptable Behavior

* Harassment
* Discrimination
* Personal attacks
* Trolling
* Hate speech
* Intimidation
* Publishing private information without permission

## Enforcement

Project maintainers are responsible for clarifying and enforcing standards of acceptable behavior.

Maintainers may remove, edit, or reject comments, commits, code, issues, and contributions that violate this Code of Conduct.

Violations may result in:

* Warning and request to correct behavior
* Temporary ban from community spaces
* Permanent ban for repeated or severe violations

## Scope

This Code of Conduct applies to:

* GitHub issues, discussions, and pull requests
* Project chat and community forums
* Public representation of the project (talks, social media, events)

## Reporting Issues

Report unacceptable behavior to:

[httpforge@outlook.com](mailto:httpforge@outlook.com)

All reports will be reviewed confidentially.
